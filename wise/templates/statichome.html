Wise is a web framework for doing simple symbolic calculations and sharing web documents with mathematical content.
Try online
Watch the Screencast
Functional Backend

The mathematical logic in Wise is all written in the functional programming language Pure, a dynamically typed language with Haskell style syntax and native term rewriting functionality.

Two Dimensional Interaction

Mathematical expressions can be entered and manipulated using a graphical interface. The system is modeled after “Click and Prove” interfaces found in some theorem provers.

Extensible

New mathematics can be added without ever writing a single line of Python. Wise attempts to automatically build a graphical interface around new symbolic functionality.

Open Source Stack

Wise runs on top of Python/Django with the notebook written in Javascript and jQuery. MathML is used to render mathematics directly in the browser. Content can also be semantically annotated using OpenMath CDs.
